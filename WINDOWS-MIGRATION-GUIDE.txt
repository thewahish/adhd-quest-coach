STEP 4: CREATE SYNC SCRIPT FOR WINDOWS
================================================================================

Create file: D:\.Master\scripts\sync-all-projects.ps1

Content:
--------------------------------------------------------------------------------
# PowerShell script to sync all projects with GitHub
Write-Host "Syncing all projects to GitHub..." -ForegroundColor Cyan

Get-ChildItem -Directory D:\Projects | ForEach-Object {
    if (Test-Path "$($_.FullName)\.git") {
        Write-Host "=== $($_.Name) ===" -ForegroundColor Yellow
        Set-Location $_.FullName

        # Check for uncommitted changes
        $status = git status -s
        if ($status) {
            Write-Host "  Committing local changes..." -ForegroundColor Green
            git add -A
            git commit -m "Auto-sync from Windows - $(Get-Date -Format 'yyyy-MM-dd HH:mm')"
        }

        # Pull and push
        git pull --rebase origin main 2>&1 | Out-Null
        git push origin main 2>&1 | Out-Null
        Write-Host "  Done!" -ForegroundColor Green
    }
}

Set-Location D:\
Write-Host "All projects synced!" -ForegroundColor Cyan

IMPORTANT NOTES FOR CLAUDE CODE ON WINDOWS
================================================================================

1. **Git must be installed** on Windows PC
   - Download from: https://git-scm.com/download/win
   - Make sure to add to PATH during installation

2. **GitHub authentication** must be configured
   - Run: git config --global user.name "thewahish"
   - Run: git config --global user.email "your-email@example.com"
   - Use GitHub CLI (gh) or HTTPS with token

3. **PowerShell execution policy** may need adjustment
   - Run as Admin: Set-ExecutionPolicy RemoteSigned

4. **Path differences from Mac:**
   - Mac: /Volumes/Ai/Projects/
   - Windows: D:\Projects\
   - Use backslashes on Windows

5. **Keep both systems in sync:**
   - After working on Mac: sync all to GitHub
   - Before working on Windows: pull all from GitHub
   - After working on Windows: sync all to GitHub
   - Before working on Mac: pull all from GitHub
